# maze_game
maze_game for mvc

## 미로통과 게임
* 가로 세로 크기를 입력 받는다. 
* 최소 크기는 10 이며 하나의 값만 받을 경우 둘다 동일한 것으로 상정한다.
* 입력받은 크기의 블럭을 채우고 채워진 공간은 최소한 한 블럭의 폭으로 길이 만들어져야 한다.
* 사용자의 커서는 상하좌우로 이동이 가능해야하며 주어진 길로만 이동이 가능해야 한다.
* 만들어진 미로의 시작점, 종료점이 있어야 하며 사용자 커서는 시작점에서 시작이 가능해야 한다.
* 요구사항 정의서, 설계서, 시퀀스 차트 가 있어야 한다.

## 실행 방법
* pip install -e .
* python -m mazegame.main

## 삭제 방법
* pip uninstall mazegame
